# May 2026 Town Hall

React/Vite slide deck for TBDC's May 2026 Town Hall.

## Commands

- `pnpm install`
- `pnpm run dev`
- `pnpm run typecheck`
- `pnpm run build`

## Navigation

- Right / Space / Enter: next slide
- Left / Backspace: previous slide
- `T`: table of contents
- `F`: fullscreen
- Home / End: first or last slide
